硬盘对大家来说应该是再熟悉不过了存储设备了。但如果现在有1500块硬盘摆在你的面前，除了卖掉和再利用以外，你会选择怎样的方式来自娱自乐呢？

看看下面这几位吧，用1500块硬盘当作多米诺骨牌，搭建成一条特殊的硬盘路。从第一块推到的那一刻到最后落水的硬盘，场面很大，声音也很特别。

除了被硬盘多米诺骨牌震撼到之外，还感慨到作者的"豪宅"实在是太大了！
